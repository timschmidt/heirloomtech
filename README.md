heirloomtech
============

Plans and source files for heirloomtech projects

After cloning the repository, type the following to clone the various submodules:

```
$ git submodule init
$ git submodule update
```

To use OpenJSCAD:
$ yum install nodejs

- load openjscad in web browser
- dock the browser to half the screen
- drag a project folder into the openjscad window
- check the 'auto-reload' checkbox
- open the files in the editor of your choice
- dock the editor to the other half of your screen
- hack
